Tidy Task App

Stack: C#, .NET, UWP, UI Design

Repository Case Study (PDF)

Overview

Tidy Task is a task management application designed for simplicity and productivity. Users can create, edit, mark as completed, and delete tasks through an intuitive UI.

Key Features

  • Create, update, and delete tasks with input validation.
  • Local data persistence using UWP storage APIs.
  • Filter tasks by status and search by keywords.
  • Accessible design with proper contrast and keyboard navigation.

Tech Stack & Tools

  • C# and .NET for core logic.
  • UWP + XAML for UI and data binding.
  • MVVM architecture for maintainability.

Architecture

Layered architecture: UI (XAML), ViewModels, and Data Services. Tasks are serialized to JSON and stored in local app data for persistence.
